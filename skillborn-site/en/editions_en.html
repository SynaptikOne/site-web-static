GENOME_COMPRESSOR Project Editions

GENOME_COMPRESSOR is a project for open science and digital innovation, exploring the intersection of biology, mathematics, cryptography, and technology.

Open Source Edition

Published under the MIT license and available publicly on GitHub, the Open Source Edition is the scientific core of the project: reproducible, modifiable, and shareable by everyone.

  • Language: Python
  • License: MIT
  • Access GitHub Repository →

Secure Edition (Restricted Access)

An experimental extension integrating hybrid and bio-inspired encryption modules. Accessible only via restricted license or academic/industrial collaboration.

  • Languages: Rust and Python
  • Status: Private development
  • Access: By request

Request access: contactsynaptikone@gmail.com

Post-Quantum Edition (Restricted Access)

Advanced version integrating post-quantum cryptography for sensitive data. Intended for institutions and laboratories handling sensitive information.

  • Language: Rust
  • Status: Advanced prototype
  • Access: By request with specific agreement

Collaboration: contactsynaptikone@gmail.com

Scientific & Community Access

An open but controlled platform is provided for sensitive modules. Researchers, students, and engineers can submit access or collaboration requests.

Transparency & Ethical Commitment

SynaptikOne is committed to free, reproducible, and ethical science. No algorithm will be commercialized without guaranteeing public access to its scientific base.

Institutional Access Request

Reserved for recognized institutions seeking access to restricted editions.

You will receive a response within 5 to 10 working days. Access to restricted editions does not allow redistribution without explicit permission.
