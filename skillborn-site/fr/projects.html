Projets de recherche technologique

Chez SynaptikOne, nous explorons les frontières entre science, informatique et mathématiques pour concevoir des outils libres, novateurs et ouverts à la contribution.

GENOME_COMPRESSOR

Un compresseur bio-inspiré de données, fondé sur l'encodage ADN. Ce projet open-source interroge les mécanismes naturels pour inventer de nouveaux schémas de compression algorithmique.

Voir le projet sur github

SECURE EDITION (prototype en cours )

Module expérimentale de chiffrement avancé pour GENOME_COMPRESSOR, destiné aux contextes sensibles. Une version dédiée aux usages professionnels et industriels pourra être proposée ultérieurement sous conditions spécifiques.
